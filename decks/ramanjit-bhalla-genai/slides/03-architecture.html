Actual recommendation path
Implemented flow

Profile in. Eligible shortlist out. GenAI rerank on top.

This is the critical design decision: the model does not browse the full catalog. It only reasons over what the deterministic engine has already allowed.

1. Startup profile
sector, stage, team size, data handled, assets, regulations, and advanced fields.
2. Risk engine
deterministic profiling and category scores across cyber, privacy, liability, property, governance, and regulatory dimensions.
3. Eligibility engine
hard gates, compliance triggers, underwriting constraints, mandatory covers, and bundle shortlist.
4. Gemini reranker
strict JSON over startup profile, scores, shortlisted products, shortlisted bundles, and constraints.
5. Audited response
bundle match, ranked products, rationale, confidence, mode, source, fallback state, and shadow diff.
Model freedom is intentionally narrow; control stays with deterministic logic.
